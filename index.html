Un div es un espacio dentro de un archivo HTML.
Los divs parten de la base <div>( (Se abre al div) </div> (Se cierra el div). Un Div puede contener otros Div. A los divs se les puede atribuir carateristicas css, por medio de ID o Class un DIV pude tener mas de un class pero solo un ID.
